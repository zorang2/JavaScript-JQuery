# 1장 정리
1. HTML, CSS, Javascript, Jquery로 작업한 눈에 보이는 영역을 통틀어 ***Front-End 개발***이라 하고, JSP, PHP, ASP, DB로 작업한 눈에 보이지 않는 영역을 통틀어 ***Back-End개발***이라 함.
2. 자바스크립트의 가장 큰 특징은 ***객체(Object)*** 기반 언어라는 것임.
3. 크롬 개발자 도구에는 웹 개발에 유용한 패널이 있음, 이 중 HTML 요소를 검사하는 ***Element패널***과 자바스크립트 코드 파싱(parsing)과정을 보며 오류를 디버깅할 수 있는 ***Source패널***이 있습니다.
